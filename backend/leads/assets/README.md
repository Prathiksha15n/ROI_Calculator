# Static Roadmap PDF Asset

## File Location
Place your finalized roadmap PDF file here with the exact filename:

**`FSM_Roadmap.pdf`**

## Requirements
- File must be named exactly: `FSM_Roadmap.pdf`
- This is a static file that will be attached to ALL emails
- No personalization is done to this PDF
- The same file is reused for every lead

## Path
The system expects the file at:
```
backend/leads/assets/FSM_Roadmap.pdf
```

## Notes
- This file is attached to every email sent to leads
- If the file is missing, email sending will fail with a clear error message
- Do NOT delete this file after placing it here
